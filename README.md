# Hacked Site

A fun React app with a hacked theme created for Maryan and Monix.

## Getting Started

### Prerequisites
- Node.js installed
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build

To build for production:
```bash
npm run build
```

## Features
- Glitch effect title
- Flicker animations
- Interactive button
- Image gallery with hacked aesthetic
